# Nayovika Collection - Phase 1 Plan

Here is the step-by-step plan for what we are implementing after Step 2 (Tailwind CSS Installation).

Create a simple, mobile-friendly website that:

- shows your saree collection,
- displays product details like price, fabric, and length,
- allows customers to order through WhatsApp,
- links to Instagram,
- looks professional on mobile and desktop,
- is easy to expand later into full e-commerce.

before each step make sure we need modern glassmorphism  ui/ux design approach and smooth animation effects and production standard code.
